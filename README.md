# PayMob Rails
PayMob rails plugin

## Usage

To mount the engine, add this line to your routes file
```ruby
mount PaymobRails::Engine, at: '/paymob'
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'paymob_rails'
```

And then execute:
```bash
$ bundle install
```
